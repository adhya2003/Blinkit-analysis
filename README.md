# Blinkit-analysis
Here's a professional GitHub project description for your **Blinkit Power BI Dashboard**:

 Blinkit Sales Analysis Dashboard | Power BI

Developed an interactive **Power BI dashboard** to analyze Blinkit's sales performance, customer behavior, and product insights. The dashboard transforms raw sales data into meaningful visualizations, helping identify business trends and support data-driven decision-making.

🔹 Key Features

* 📊 Interactive dashboard with slicers and filters
* 💰 Total Sales, Profit, and Average Sales analysis
* 🛒 Product category and item type performance
* ⭐ Customer ratings and sales distribution
* 🏬 Outlet-wise sales analysis by size, location, and type
* 📈 Sales trends and KPI cards for quick insights
* 🎨 User-friendly and visually appealing dashboard design

 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Calculated Measures & KPIs)
* Excel/CSV Dataset

Dashboard Insights

* Identified top-performing product categories and outlets.
* Compared sales across different outlet sizes and locations.
* Analyzed customer ratings and purchasing patterns.
* Enabled stakeholders to monitor key business metrics through interactive visualizations.

🚀 Skills Demonstrated

* Data Cleaning
* Data Modeling
* DAX Calculations
* Data Visualization
* Dashboard Design
* Business Intelligence
* Analytical Thinking

**Project Type:** Business Intelligence | Data Analytics | Power BI Dashboard
